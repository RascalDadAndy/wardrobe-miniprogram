# 智能穿搭助手 - 部署指南

## 问题解决方案

你遇到的错误 `Cannot read property 'callFunction' of undefined` 是因为云开发没有正确初始化。

我已经为你修改了代码，现在支持**两种模式**：
1. **云开发模式** - 完整功能，需要开通云开发
2. **本地模式** - 无需配置，数据保存在本地，适合快速体验

---

## 方案一：本地模式（推荐，5分钟搞定）

这个方案**不需要开通云开发**，可以直接运行。

### 步骤：

1. **直接导入项目**
   - 打开微信开发者工具
   - 点击"导入项目"
   - 选择 `wardrobe-miniprogram` 文件夹
   - AppID 选择"测试号"
   - 点击"确定"

2. **完成！**
   - 小程序会自动运行在本地模式
   - 数据会保存在手机本地
   - 可以正常体验所有基础功能

### 本地模式限制：
- ❌ 数据不能跨设备同步
- ❌ 不能使用虚拟试衣功能（需要AI接口）
- ❌ 图片只能本地查看
- ✅ 衣柜管理、上传、查看功能正常

---

## 方案二：云开发模式（完整功能）

### 步骤 1：开通云开发

1. 打开微信开发者工具
2. 点击顶部工具栏的 **"云开发"** 按钮
3. 点击"开通"
4. 选择"免费版"套餐
5. 等待开通完成

### 步骤 2：获取环境ID

1. 在云开发控制台顶部，复制**环境ID**
   - 格式如：`wardrobe-assistant-xxx`

### 步骤 3：修改配置文件

打开 `app.js` 文件，修改第6行：

```javascript
globalData: {
  userInfo: null,
  openid: null,
  cloudEnv: 'wardrobe-assistant-xxx',  // ← 改成你的环境ID
  cloudInitialized: false,
  // ...
}
```

### 步骤 4：指定云环境

**方法A - 通过界面指定：**
1. 在微信开发者工具中，点击菜单栏的 **"设置"** → **"项目设置"**
2. 找到"云开发"区域
3. 在"云环境"下拉框中选择你的环境

**方法B - 修改 project.config.json：**

打开 `project.config.json` 文件，添加：

```json
{
  "appid": "你的AppID",
  "projectname": "wardrobe-assistant",
  "cloudfunctionRoot": "cloudfunctions/",
  "cloudfunctionDefaultEnv": "你的环境ID",  // ← 添加这一行
  // ...
}
```

### 步骤 5：部署云函数

1. 在左侧文件树中找到 `cloudfunctions` 文件夹
2. 展开后看到5个云函数：
   - `login`
   - `uploadImage`
   - `generateTryon`
   - `getSuggestion`
   - `generateOutfit`

3. **逐个右键点击**每个云函数
4. 选择 **"创建并部署：云端安装依赖"**
5. 等待部署完成（每个约10-20秒）

### 步骤 6：初始化数据库

1. 点击顶部的 **"调试器"** 标签
2. 在 Console 控制台粘贴以下代码：

```javascript
wx.cloud.callFunction({
  name: 'login',
  data: {}
}).then(res => {
  console.log('云开发初始化成功', res);
}).catch(err => {
  console.error('初始化失败', err);
});
```

3. 按回车执行
4. 看到"云开发初始化成功"即可

### 步骤 7：运行项目

1. 点击工具栏的 **"编译"** 按钮
2. 小程序现在运行在云开发模式

## 常见问题

### Q1: 还是提示 "callFunction of undefined"
**解决**：
1. 确认 `app.js` 中的 `cloudEnv` 已修改
2. 确认已在"设置"中选择了云环境
3. 重启微信开发者工具
4. 重新导入项目

### Q2: 云函数部署失败
**解决**：
1. 确认已开通云开发
2. 确认已选择云环境
3. 检查网络连接
4. 尝试重启开发者工具

### Q3: 提示 "AppID 不合法"
**解决**：
- 使用测试号（选择"测试号"选项）
- 或去 https://mp.weixin.qq.com 注册小程序获取真实 AppID

### Q4: 如何切换到本地模式？
**解决**：
- 什么都不用做！代码会自动检测
- 如果云开发初始化失败，会自动切换到本地模式
- 会看到提示"已切换到本地存储模式"

### Q5: 本地模式数据在哪里？
**解决**：
- 数据保存在微信本地存储
- 路径：手机文件管理 → Android/data/com.tencent.mm/MicroMsg/...
- 清理缓存会丢失数据
