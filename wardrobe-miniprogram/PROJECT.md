# 智能穿搭助手 - 微信小程序项目文档

## 项目概述

"智能穿搭助手"是一款基于微信小程序的智能穿搭管理应用，帮助用户：
- 数字化管理个人衣柜
- 通过AI生成虚拟试衣效果
- 获取个性化的穿搭建议

## 核心功能

### 1. 衣柜管理
- 拍摄或上传衣物照片
- 自动分类（上衣、裤子、裙子、外套、鞋子、配饰）
- 标签管理和智能搜索
- 收藏和批量管理

### 2. 虚拟试衣
- 上传个人照片
- 选择衣物进行虚拟试穿
- AI生成逼真的穿搭效果图
- 保存和分享试衣结果

### 3. 穿搭建议
- 根据场合、季节获取搭配建议
- AI点评和风格分析
- 色彩搭配建议
- 基于已有衣物的搭配方案

## 快速开始

### 1. 导入项目
```bash
# 将项目文件夹导入微信开发者工具
# 选择 wardrobe-miniprogram 目录
```

### 2. 配置云开发
- 开通云开发服务
- 记录环境ID
- 修改 app.js 中的 cloudEnv

### 3. 初始化数据库
```javascript
// 调用云函数初始化数据库
wx.cloud.callFunction({
  name: 'initDatabase',
  data: { action: 'initAll' }
})
```

### 4. 部署云函数
- 右键云函数文件夹
- 选择"创建并部署：云端安装依赖"

### 5. 运行项目
- 点击"编译"按钮
- 使用真机调试或模拟器预览

## 功能扩展

### 添加新的衣物分类
1. 修改 `app.js` 中的 `clothingCategories`
2. 更新 `pages/upload/upload.js` 中的分类选项

### 自定义配色方案
1. 修改 `app.wxss` 中的CSS变量
2. 更新各页面的样式文件

### 集成更多AI功能
1. 在 `utils/ai.js` 中添加新函数
2. 创建对应的云函数
3. 在页面中调用

## 注意事项

1. **图片存储**：建议使用云存储，注意配置存储权限
2. **API调用**：生图API有调用限制，建议添加使用次数控制
3. **数据安全**：数据库权限配置为仅创建者可读写
4. **性能优化**：图片上传前建议压缩，列表使用分页加载

## 常见问题

### Q: 云开发环境如何开通？
A: 在微信开发者工具中点击"云开发"按钮，按提示开通即可。

### Q: 如何修改配色方案？
A: 修改 `app.wxss` 中的CSS变量，所有页面会自动生效。

### Q: 如何添加新的衣物属性？
A: 修改 `database/schema.md` 中的字段定义，并更新上传页面。

### Q: 虚拟试衣效果不理想？
A: 确保用户照片为正面全身照，衣物图片背景简洁。
